# 百度首页复刻请求

## 用户请求

- 原始网址：`https://www.baidu.com`
- 复刻范围：搜索输入框、搜索按钮点击交互、搜索结果文案展示、结果列表翻页功能。

## 已确认复刻方案

用户已确认阶段一方案，可以进入 project 初始化。

## 复刻目标

| 项目 | 内容 |
| --- | --- |
| Target ID | `baidu` |
| Project 路径 | `projects/baidu/` |
| 复刻页面访问地址 | `http://127.0.0.1:5173/replica/baidu` |
| 评估配置路径 | `projects/baidu/config/target.json` |
| 推荐评估模式 | `交互辅助评估` |

## 页面范围

- 百度首页初始态。
- 搜索结果展示态。
- 搜索结果翻页态。

## 状态范围

- `home`：首页初始态。
- `search-results`：输入关键词并搜索后的结果状态。
- `pagination`：搜索结果翻页后的状态。
- `empty-query`：空输入搜索反馈，作为交互完整性补充状态。

## 功能范围

- 搜索输入框输入关键词。
- 点击搜索按钮触发搜索。
- 按 Enter 触发搜索。
- 展示搜索结果文案。
- 展示结果列表。
- 支持结果列表翻页。

## 不复刻范围

- 不调用百度真实搜索接口。
- 不实现真实登录、账号菜单、广告投放、个性化推荐。
- 不复刻移动端，仅复刻桌面端。
- 搜索结果数据使用本地 mock，不保证与真实搜索结果内容一致。

## 原站验证规则

- 真实页面无法确认、出现验证码/安全验证、要求登录、进入错误页或缺少必需状态时，立即暂停并请求用户介入。
- 空白页、错误页、验证码页、安全验证页不得作为原站基线。
- 必需状态未采集完成前不得开始复刻实现。
